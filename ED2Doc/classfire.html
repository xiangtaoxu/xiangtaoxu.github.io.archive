Detailed Description

This module contains routines to obtain fire disturbance rates.

These subroutines are intended to calculate fire intensity and burned area which are or can be used to obtain fire disturbance rate and survivorship

Author
Paul Moorcroft, converted to fortran by David Medvigy
10 Jan 2018. MLO converted it into module so the code compiles with ifort 17. Also implementing process-based model, step by step.
